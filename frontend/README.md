# QuickAppointment

## Description

QuickAppointment is a web-based appointment booking system that allows users to easily schedule appointments online. Users can register, log in, view available time slots, and book appointments. Administrators can manage bookings, users, and service schedules through an admin dashboard.

## Features

* User registration and login
* Online appointment booking
* View available time slots
* Appointment history
* Admin dashboard
* Manage users and bookings

## Technologies Used

* Frontend: React.js
* Backend: Node.js (Express.js)
* Database: MySQL

## Installation

### 1. Clone the repository

```
git clone https://github.com/your-username/quickappointment.git
```

### 2. Install frontend dependencies

```
cd frontend
npm install
```

### 3. Install backend dependencies

```
cd backend
npm install
```

### 4. Configure database

Update database connection details in the backend configuration file.

### 5. Run backend server

```
npm start
```

### 6. Run frontend application

```
npm start
```

### 7. Open in browser

```
http://localhost:3000
```
